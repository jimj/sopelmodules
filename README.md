Sopel plugins
=============
Customization of Sopel.  Includes MongoDB support.

#### Requirements ####
* MongoDB 3.x
* PyMongo 3.x
* Sopel
* BeautifulSoup 4

#### Installation ####
These plugins assume MongoDB is available with a database named 'ircbot'.

* Run sopel
* git clone this project into ~/.sopel/modules
* .reload or restart sopel
